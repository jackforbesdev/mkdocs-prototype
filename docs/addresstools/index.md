# AddressTools

Welcome to AddressTools documentation. AddressTools is a comprehensive address verification and validation solution for Salesforce that ensures your address data is accurate, standardised, and up to date.

## Overview

AddressTools enhances your territory management, reduces delivery failures, simplifies reporting, and improves data quality through global address verification and validation. Whether you need to verify addresses at point of entry or clean existing data, AddressTools has you covered.

## Key Features

- **Global Address Verification**: Verify addresses against postal authorities worldwide (USPS, Canada Post, Royal Mail, etc.)
- **Geocoding**: Add latitude and longitude coordinates to verified addresses for location-based workflows
- **Advanced Validation Rules**: Flexible country, state, and ZIP code validation rules
- **Country & State Standardisation**: Standardise country and state values to your preferred format
- **Flow Component**: Native Salesforce Flow component for address verification in workflows
- **Interactive Data Entry**: ZIP code lookup, country/state suggestions, and autocomplete features
- **US County Data**: Autopopulate accurate county values using ZIP+4 codes
- **Enhanced Address Editor**: Lightning component for viewing and editing addresses
- **Custom Objects Support**: Verify addresses on custom objects and fields
- **Per-Profile Configuration**: Customise settings for different users and profiles
